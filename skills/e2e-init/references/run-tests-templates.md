# run-tests.sh Templates

Templates emitted by `/e2e-init` at Step 5. Write the content of the relevant
template verbatim into `scripts/run-tests.sh`, then adapt the `# REPLACE:` /
`# ADD:` placeholder comments using the repo-specific information gathered in
Steps 3 and 4.

---

## Firmware Static-Analysis Backbone

Used when the project type (Step 2) is **Firmware**. Runs cppcheck and
clang-tidy only when they are present on PATH; exits 0 with a clear message
when neither is available so the pre-commit hook never hard-fails on a machine
without those tools installed.

```bash
#!/bin/bash
# scripts/run-tests.sh
# Generated by /e2e-init — Firmware static-analysis backbone.
# No host test runner is available for this embedded firmware repo.
# For HIL / simulation test sequences, see plans/e2e-plan.md.
# Usage: bash scripts/run-tests.sh

set -uo pipefail

RC=0
SA_RAN=false

if command -v cppcheck &>/dev/null; then
  echo "Running cppcheck..."
  cppcheck --error-exitcode=1 --quiet . && echo "cppcheck: OK" || RC=1
  SA_RAN=true
fi

if command -v clang-tidy &>/dev/null; then
  echo "Running clang-tidy (dry-run check)..."
  # REPLACE: adjust file selection / compile-commands path for this repo
  # (prefer compile_commands.json when present).
  files=$(find . -name '*.c' -o -name '*.cpp')
  if [ -n "$files" ]; then
    printf '%s\n' "$files" | xargs clang-tidy --quiet || RC=1
  fi
  SA_RAN=true
fi

if [ "$SA_RAN" = false ]; then
  echo "No host test runner for firmware — see plans/e2e-plan.md HIL plan."
  echo "(Install cppcheck or clang-tidy to enable static-analysis checks here.)"
  exit 0
fi

exit "$RC"
```

## Non-Firmware Host-Runner

Used when the project type (Step 2) is **Backend**, **UI/Client**, or the
host-testable portion of a **Mixed** repo. Reads service boundaries from
`.claude/testing-plan.md`, maps changed files to services, and runs only the
affected service's tests (or the full suite when a trigger path changes).

  "src/utils/"

  echo "Running full test suite..."
  # REPLACE: insert the full suite run command for this repo
